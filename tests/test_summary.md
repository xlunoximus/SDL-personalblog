# Отчет о тестировании — PersonalBlog

## 1. Общая информация

Данный документ содержит результаты автоматизированного тестирования backend-части приложения PersonalBlog.

Все основные модули приложения проверены с использованием JUnit 5 и Mockito.

## 2. Компоненты системы

| Компонент        | Технология                    |
| :--------------- | :---------------------------- |
| Backend          | Spring Boot 3.x (Java 17+)    |
| База данных      | H2 (для тестов), PostgreSQL/MySQL (prod) |
| Аутентификация   | JWT                           |
| ORM              | JPA / Hibernate               |
| Frontend         | HTML5, CSS3, JavaScript (ES6+), Thymeleaf |
| Тестирование     | JUnit 5, Mockito, MockMvc     |
| Сборка           | Maven 3.9+ / Gradle           |

## 4. Основные достижения

*   Высокое покрытие тестами модулей `PostService`, `CommentService`, `UserService`.
*   Валидация корректности аутентификации и авторизации через JWT.
*   Проверка всех контроллеров (`PostController`, `CommentController`, `AuthController`) на корректность возврата представлений и HTTP статусов.
*   Тестирование всех основных моделей данных (`User`, `Post`, `Comment`).
*   Успешная проверка CRUD операций для постов.
*   Валидация входных данных (например, проверка длины пароля, непустого заголовка поста).
